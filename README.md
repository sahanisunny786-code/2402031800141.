🌌 Neon Weather App – TechnoHacks Project
The Neon Weather App is a modern, visually appealing weather application built using HTML, CSS, and JavaScript, designed with a futuristic neon-glass UI. This project allows users to quickly search for any city and view real-time weather information in an elegant and interactive interface.
The application integrates with the OpenWeatherMap API to fetch live weather data, ensuring accurate and up-to-date information. Users can simply enter a city name, and the app dynamically displays key weather details such as temperature, weather condition, humidity, and wind speed.
One of the main highlights of this project is its glassmorphism design with neon effects, which creates a smooth and immersive user experience. The animated glowing background and soft UI transitions enhance the overall aesthetics, making it more engaging than traditional weather apps.
✨ Features
🔍 Search weather by city name
🌡️ Real-time temperature display (in Celsius)
🌤️ Dynamic weather condition icons
💧 Humidity and 🌬️ wind speed details
❌ Error handling for invalid city inputs
🎨 Neon glass UI with smooth animations
🛠️ Technologies Used
HTML5 – Structure of the application
CSS3 – Styling with glassmorphism and animations
JavaScript (Vanilla JS) – Logic and API integration
OpenWeatherMap API – Real-time weather data
🚀 How It Works
When a user enters a city name and clicks the search button, the app sends a request to the OpenWeatherMap API. The response is processed and displayed dynamically on the UI. Based on the weather condition, the app also updates the weather icon to match the current climate.
